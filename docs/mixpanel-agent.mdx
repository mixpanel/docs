## Overview

Mixpanel Agent is an AI analyst built into Mixpanel. Ask it questions or give it tasks and the Agent interprets your intent and selects the right tools automatically. A single prompt can run an analysis, build a report, create a cohort, or walk you through a user's session replays without you specifying each step.

## Core Capabilities

The Mixpanel Agent has access to a broad set of Mixpanel capabilities and selects tools automatically based on what you ask.

| Capability | What the Agent can do |
| --- | --- |
| Report Creation & Editing | Create new Insights, Funnels, Retention, and Flows reports from a description. Adjusts existing report metrics, filters, breakdowns, and time ranges. |
| Boards Creation & Editing | Create new Boards or use the Agent to edit an existing Board's layout and content |
| Metrics | Define metrics in plain language and save them for reuse across your project |
| Cohorts | Build behavioral cohorts from a description and save them for analysis or targeting |
| Search project data | Search across your project's events, properties, saved reports, Boards, feature flags, and experiments to find what already exists before building something new |
| Analyze reports & Boards | Interpret, explain, and investigate any report or Board. Automatically picks up the context of whatever you currently have open, so you don't need to describe your setup |
| Session replay analysis | From a prompt or existing playlist, Mixpanel curates relevant replays, analyzes them alongside your events data, summarizes findings for follow-up questions, and surfaces timestamps for key moments |
| Experiments & Feature Flags | Reference experiments and feature flags in any investigation, interpret experiment results in plain language, and create new experiments or flags through a conversational flow |
| Mixpanel Docs | Answer questions about how Mixpanel features work |
| Root Cause Analysis | Run a root cause analysis for a metric change. [See Root Cause Analysis for details](/docs/root-cause-analysis) |

**Working on an analysis?** The Agent automatically picks up your current page as context. Ask it to explain a trend, dig into an anomaly, or modify what's in front of you. No need to describe what you're looking at first.

## Access

Where you'll find the Mixpanel Agent:

- **Side navigation:** Click Mixpanel Agent to open a full-page experience
- **Contextual panel:** Click "Open in side view" to open the Agent in a side panel so you can bring the Agent with you as you navigate Mixpanel
- **Create New:** Click Mixpanel Agent from the Create New menu

## Example Use Cases

### Explore a direction

You have a theory or idea. The Mixpanel Agent helps you test it quickly.

_Does our new onboarding flow actually improve activation?_

_I think users who invite a teammate in their first week retain better. Can you validate that?_

### Put your ideas into action

You've landed on a direction. Now ship the change and measure the impact.

_Set up an experiment to test a simplified checkout against the current flow_

_Roll out the new pricing page to 10% of users behind a feature flag_

### Investigate what happened

Something moved in your metrics. The Agent helps you get a clear picture to get to the bottom of it.

_Explain this spike in signups_

_Did anything unusual happen with our checkout flow_

### Get up to speed fast

You need context before a meeting or decision. The Agent searches what already exists in your project so you're not starting from scratch.

_Show me our top of funnel KPIs._

_Summarize this analysis for me before my meeting_

_What tracking do we have on our messaging feature_

### Build your analysis

Build out boards, reports, metrics, and cohorts to share across your team for analysis.

_Make a Board for our weekly growth review_

_Save our activation definition_

_Create a cohort of power users so we can target them in experiments_

### Understand your users

Dive deeper to understand what individual users or segments actually experienced.

_Find sessions where users hit an error during checkout, walk me through what they did, and surface timestamps of key moments_

_This user churned after 3 days. Walk me through their sessions_

## Business Context

The Mixpanel Agent uses your project's [Business Context](/docs/business-context) to give more relevant answers. The more context you provide, the better the Agent's answers.

## Verified Mode

Admins can control what data the Agent surfaces by enabling Verified Mode in Project Settings. When enabled, Mixpanel AI will only use [Verified Data](/docs/data-governance/lexicon#verify-data). Verified Data is events, metrics, behaviors, and cohorts that your team has explicitly reviewed and verified in Lexicon. This ensures AI responses are grounded in data your organization trusts.

When unverified data is excluded from a response, users are notified that additional relevant data exists but wasn't included.

## Sharing

To share a Mixpanel Agent chat, click on the chat name and select **Share Link.**

<Frame>
  ![Agent Share Link](/images/image-1.png)
</Frame>

Shared chats are view-only. Anyone with the link who has access to the project can view the chat.

<Frame>
  ![Agent Chat Sharing](/images/image.png)
</Frame>

The link shows the chat as it existed when you generated it, up to the most recent message at that time.

Optionally, you can share the chat along with the current page content. This is useful if the context of your chat is about a Board, Report, Experiment, or Session Replay Playlist.

To make a chat private again, select **Keep Private**. Existing share links stop working once the chat is private.

## Chat History

Access more options for managing chats from the chat header, including: 

- **New Chat:** Click the pencil icon to start a new chat. When you start a new chat, your previously active chat is still available in history.
- **Rename chat:** Change the title of a chat
- **Delete chat:** Remove a chat from history
- **Chat history:** Access all your previous chats when working in the side panel. If you're viewing the Agent as a full page you will see the chat history on the left automatically.
