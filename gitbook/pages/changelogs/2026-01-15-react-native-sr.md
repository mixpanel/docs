# 📱 Session Replay is now available for React Native
_2025-01-15_

![](.gitbook/assets/rn-sr-ga.png)

Now you can replay user sessions for React Native apps, bringing Session Replay to your cross-platform mobile development workflow. With React Native Session Replay, you can:

- Answer the what and why in one place by moving seamlessly between reports and replays
- Find what matters faster by filtering replays with trusted Mixpanel events, properties, or cohorts
- Watch the complete user journey across your React Native iOS and Android apps

Session Replay is free to try. Check out the resources below to get started!

[View Docs →](https://app.gitbook.com/s/qGpd1uH02qXOCsOiKqLX/session-replay/implement-session-replay)
