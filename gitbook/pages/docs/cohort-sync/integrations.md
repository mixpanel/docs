# Integrations: Export your cohorts to a third-party tool

{% hint style="info" %}
This page highlights integrations that sends Mixpanel Cohorts to external
  destinations. Learn more about [integrations that imports data from other
  destinations into Mixpanel](../tracking-methods/integrations.md) and
  [integrations that exports data from Mixpanel to data
  warehouses](../data-pipelines/integrations.md).
{% endhint %}

These integrations allow you to send your [Cohorts](../users/cohorts.md) to other destinations for audience targeting. You may review our complete integrations directory [here](https://mixpanel.com/partners/integrations). If you don't see what you're looking for [reach out to us](https://mixpanel.com/get-support).
