title: "Core Reports"
---

The Fundamentals module teaches you Mixpanel's core reports and how to use them. Whether you're new to Mixpanel or building on what you already know, you'll come away able to analyze user behavior with confidence.

The module is hosted on [Mixpanel University](https://university.mixpanel.com/student/path/3411736-mixpanel-fundamentals). By the end, you'll be able to:

- Use Mixpanel's core reports: Insights, Funnels, Flows, and Retention
- Analyze user behavior
- Measure what matters for your product
