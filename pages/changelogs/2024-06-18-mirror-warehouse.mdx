Introducing Mirror sync mode in Warehouse Connectors. This allows modifications to previously sent rows in Warehouse tables to be accurately reflected in Mixpanel. This unlocks accurate analysis for all data types, even data that changes such as transactions.

This is available in Mixpanel Connectors for BigQuery and Snowflake event tables. Warehouse Connectors requires a paid add on for Enterprise Plans.
