# Setting up the Mixpanel SDK with Wordpress

There are 2 ways to add Mixpanel with Autocapture to Wordpress:

**Option 1 — Via plugin**

You can use either the free [Insert Headers and Footers](https://wordpress.com/plugins/insert-headers-and-footers) plugin or, if you’re already using Google Tag Manager, you can add the snippet via Google Tag Manager [link to GTM doc].

If you use a plugin:

1. Start by installing the plugin
2. Add the snippet to the header right before the closing </head> tag

**Option 2 — Edit your theme’s functions file directly**
The most straightforward way to add Mixpanel with Autocapture to Wordpress is to add it to a Theme functions file.

First, find your theme's `functions.php` file. This can be found either in `app/public/wp-content/themes/<your-theme>` folder or in your WordPress admin under **Tools** -> **Theme Filter Editor**.

Next, create an `add_mixpanel` function with your snippet like this:
