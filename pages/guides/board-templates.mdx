<details style={{ cursor: 'pointer', marginBottom: '16px' }}>
  <summary style={{ fontSize: '1.15rem', paddingBottom: '4px' }}>
    <strong>How do I use these templates in my own project?</strong>
  </summary>
  <div style={{ paddingLeft: '24px', paddingTop: '8px' }}>

    <ul style={{ listStyleType: 'disc', paddingLeft: '20px', marginTop: '12px' }}>
      <li style={{ marginBottom: '8px' }}>
        <strong>The Template Selector:</strong> Click the <a href="https://mixpanel.com/project/?show-template-selector=true" target="_blank" style={{ color: '#b794f4', fontWeight: 600 }}>Ready to build? Create a Board &rarr;</a> button to launch the configuration wizard.
      </li>
      <li>
        <strong>From a New Board:</strong> Create a new Board (via <strong>+ Create New</strong> &gt; <strong>Board</strong>) and select a framework from the <strong>Start from a Template</strong> section.
      </li>
    </ul>
  </div>
</details>

<details style={{ cursor: 'pointer', marginBottom: '16px' }}>

  <div style={{ paddingLeft: '24px', paddingTop: '8px' }}>

  </div>
</details>
