# Board Templates

Discover Mixpanel’s core features through pre-built dashboard templates. 
Click any card to start a project with that template, or click **PREVIEW BOARD** to explore a live interactive demo. [cite: 2]

{/*
  Cards API (examples):

  {
    title: 'Company KPIs',                      // Main title for the template card
    description: 'Measure product health...',   // Supporting text explaining the use case (matches "blurb" in tours)
    img: '/templates/Company KPIs.webp',        // Image src (314x139 pixels recommended)
    previewUrl: 'https://...',                  // URL for the iframe modal (Preview)
    createUrl: 'https://...'                    // URL for the main card click (Create flow)
  }

  Notes:
  - The PREVIEW BOARD badge opens the preview link in an iframe modal for a consistent experience.
  - Clicking the card body navigates the current tab to the creation flow via useRouter.
  - Images should live under /public/templates/ for easy referencing. 
*/}
